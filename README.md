This application is a date tracker.

It has rows for hours in the day from 7AM to 5pm.

Each hour has a text area that allows the user to type in their schedule items for that hour.

The user can then save the items to their schedule using the save button to the right of the item. 

The schedule is tied to real time. It displays the past, present, and future with different colors.

The Current date is displayed at the top of the page.
